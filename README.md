VueJS

Siglas
======

Comandos
========

Cria um projeto react apenas para o front-end (Vue CLI required).
```
npx vue create [nome do app]
```

Abrir administação visual de projetos (Vue CLI required).
```
npx vue ui
```

Inicia servidor de desenvolvimento
```
npm run serve
```
